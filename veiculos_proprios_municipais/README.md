Página desenvolvida para organizar os veículos municipais do transporte escolar do município, seguindo o padrão do TCE. Projeto temporário, criado para suprir a ausência de uma empresa terceirizada.

Neste projeto organizei o HTML da forma mais semântica possível, utilizando header, main e footer. As informações foram estruturadas em tabelas, por meio das tags table, para facilitar a visualização e organização dos dados. Abaixo de cada tabela as imagens dos veículos próprios.

Também utilizei media queries para garantir a responsividade em telas menores, incluindo barra de rolagem horizontal em dispositivos com resolução reduzida, seguindo o padrão recomendado para tabelas.

Link do projeto no github pages: https://michlsilvala.github.io/veiculos_proprios_municipais/
